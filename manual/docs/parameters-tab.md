# ② Parameters tab

The Parameters tab controls the *modelling* and *evaluation* settings: which Maxent feature classes to allow, the regularization multiplier, the spatial cross-validation method, and where to write outputs. Defaults follow established Maxent practice; this chapter explains when to override them.
