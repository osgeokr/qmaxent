# QMaxent 매뉴얼

QMaxent 매뉴얼에 오신 것을 환영합니다. QMaxent는 [elapid](https://github.com/earth-chris/elapid) Python 라이브러리를 QGIS에 통합해, 모델 학습부터 공간 교차검증·Jackknife 변수 중요도·서식적합도 투영·우선조사 후보지 선정까지 Maxent 종 분포 모델링(SDM) 전 과정을 한국어/영어 UI로 제공합니다.
