# QMaxent 매뉴얼

QMaxent에 오신 것을 환영합니다 — QGIS 안에서 Maxent 종 분포 모델링을
가능하게 해주는 플러그인입니다. 본 매뉴얼은 첫 설치부터 발표 가능한
서식적합도 지도와 현장 조사 후보지 추출까지 전 워크플로를 안내합니다.

## 표기 관습

매뉴얼 전체에서 다음 관습을 사용합니다:

- **`코드 글꼴`** — 파일 이름, 메뉴 항목, 명령
- *기울임* — 종 학명과 새로운 용어
- **굵게** — 도크 컨트롤(예: **▶ Run Maxent**)
- **!!! note** 블록 — 보충 정보
- **!!! tip** 블록 — 권장 관행
- **!!! warning** 블록 — 흔히 빠지는 함정
